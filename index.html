COMO USAR

Publicação segura no GitHub

1. Mantenha este arquivo na raiz com o nome exato index.html.

2. Não envie arquivos chamados Index.html com I maiúsculo.

3. Não salve páginas do ChatGPT como HTML para usar como portal.

4. Os agentes devem ficar dentro de agentes/nome-do-agente/.

5. Os manuais devem ficar junto do respectivo agente ou em uma subpasta outputs/.
